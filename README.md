# English-Phonics
為了讓台灣小孩快速記憶，用注音和英文連結

總共涵蓋了 6 大類、70+ 張字卡，全部對照注音符號：

短母音 — a / e / i / o / u（5張）
長母音 — a_e / i_e 等魔e規則（5張）
母音組合 — ai, ay, ee, oo, ou, oi, aw 等（16張）
R控制音 — ar, er, ir, or, ur（5張）
子音組合 — sh, ch, th, ph, ng, kn, wr 等（13張）
子音連綴 — bl, cr, str, spr 等（24張）

使用方式：點字卡翻面看注音 → 用左右按鈕換張 → 切「總覽模式」可以一次看整類。
